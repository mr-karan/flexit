No hidden surprises

The motive behind creating another flexbox grid is precisely this. Flexbit doesn't make any preassumptions and no hacky code. As a result, you take full control over how the columns and row will be aligned and can even control the spacing between them.

Auto width Columns

You can add as many columns as you want in a row. Don't specify any fixed size of the columns and the grid will fit these auto width columns based on the size available in the row.

Flexbox Utilities

Flexbit implements a set of useful utilities present in Flexbox as helper classes. You can make the use of them to accelerate your development
